ChatGPT said:

To display an XML feed in an <iframe> app on Screenly.io, youll need to convert that XML into a web-friendly formatideally HTMLbecause:

An <iframe> can only display content that a browser can render (like HTML pages), not raw XML.

Here's how to make it work:

OPTION 1: Use a public XML-to-HTML converter page (easy, no-code)

  1. Find a service like rss2html.com, FeedBurner, or even host your own basic script that takes your XML URL and renders it as a readable HTML page.

  2. Take the URL of that rendered HTML page and paste it into your Screenly iframe app.

OPTION 2: Host your own HTML converter (flexible)

If you're comfortable with basic HTML + JavaScript, you can:

Step 1: Create a simple HTML file that fetches and parses the XML

html
<!DOCTYPE html>
<html>
<head>
  <title>XML Feed Viewer</title>
</head>
<body>
  <div id="output">Loading feed...</div>

  <script>
    const xmlURL = 'https://example.com/your-feed.xml'; // Replace with your XML link

    fetch(xmlURL)
      .then(response => response.text())
      .then(data => {
        const parser = new DOMParser();
        const xml = parser.parseFromString(data, "text/xml");
        let items = xml.getElementsByTagName("item"); // For RSS feeds
        let html = "<ul>";
        for (let i = 0; i < items.length; i++) {
          const title = items[i].getElementsByTagName("title")[0].textContent;
          const link = items[i].getElementsByTagName("link")[0].textContent;
          html += `<li><a href="${link}" target="_blank">${title}</a></li>`;
        }
        html += "</ul>";
        document.getElementById("output").innerHTML = html;
      })
      .catch(error => {
        document.getElementById("output").innerHTML = "Error loading feed.";
        console.error(error);
      });
  </script>
</body>
</html>

Step 2: Host that HTML file somewhere accessible (like GitHub Pages, Netlify, or your internal server).

Step 3: Use the public URL of that hosted HTML in your Screenly iframe.

Notes:

  • Make sure your XML source has CORS enabled. If not, Screenly or your browser wont be able to fetch it.

  • If it doesn't, youll need to use a proxy server or set up a server-side script that fetches the XML and outputs HTML.

Perfect! You're working with an Atom XML feed from the National Weather Service, filtered for zone ALC033 (Colbert County, Alabama). To show this feed in a Screenly iframe, you'll need a simple HTML wrapper that:

  1. Fetches the Atom feed.

  3. Displays it in a way that Screenly can render.

Here's a ready-to-go HTML viewer for that feed:

html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Weather Alerts - Colbert County, AL</title>
  <style>
    body { font-family: sans-serif; background: #111; color: #fff; padding: 20px; }
    h2 { font-size: 1.5em; }
    a { color: #00f2ff; text-decoration: none; }
    a:hover { text-decoration: underline; }
    .alert { margin-bottom: 1em; border-bottom: 1px solid #444; padding-bottom: 1em; }
  </style>
</head>
<body>
  <h1>Colbert County Weather Alerts</h1>
  <div id="alerts">Loading...</div>

  <script>
    const feedUrl = 'https://api.weather.gov/alerts/active.atom?zone=ALC033';

    fetch(feedUrl)
      .then(response => response.text())
      .then(str => new window.DOMParser().parseFromString(str, "text/xml"))
      .then(data => {
        const entries = data.getElementsByTagName("entry");
        const container = document.getElementById("alerts");

        if (entries.length === 0) {
          container.innerHTML = "<p>No active alerts.</p>";
          return;
        }

        let html = '';
        for (let i = 0; i < entries.length; i++) {
          const title = entries[i].getElementsByTagName("title")[0].textContent;
          const summary = entries[i].getElementsByTagName("summary")[0].textContent;
          const link = entries[i].getElementsByTagName("link")[0].getAttribute("href");

          html += `
            <div class="alert">
              <h2>${title}</h2>
              <p>${summary}</p>
              <a href="${link}" target="_blank">View full alert</a>
            </div>
          `;
        }

        container.innerHTML = html;
      })
      .catch(err => {
        document.getElementById("alerts").innerHTML = "Failed to load alerts.";
        console.error("Error loading Atom feed:", err);
      });
  </script>
</body>
</html>

    • Or even a Raspberry Pi if you're running a local Screenly setup
